Selenium Sniper

Selenium Sniper is a Python program (will be Java-based soon) that automatically signs you up for pre-chosen Payton enrichments at midnight, ten days in advance (the earliest possible time to sign up for enrichments). Well, technically the "automatically at midnight" part is handled by your OS's task scheduler program, but the Java program will just sit there all the time.

How it works

Selenium doesn't actually have an API, so the mobile site is used (for its simplicity) to log in to the appropriate page at http://selenium.wpcp.org/m/ then actually sign up for the enrichment.

Why Selenium?

It's not actually my choice of name, for the most part. "Selenium" is the element whose atomic number is Payton's number, 34. Selenium Sniper has nothing to do with the Selenium WebDriver system.

Where's the Java port, man?

It's on the way. Originally, I wrote Selenium Sniper in Python, because that's what I knew. The Python version works, (except for installation on Windows lol), but I want something more portable. Hopefully, writing a GUI app in Java SE will mean that I could easily port it to Android, and from there I could (again, hopefully) use J2ObjC to port it to an iOS app.
